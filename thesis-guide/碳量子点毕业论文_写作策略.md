# 碳量子点的创新制备与应用 — 毕业论文写作策略

## 一、现状分析与应对思路

### 客户提供的材料
| 材料 | 数量 | 用途 |
|------|------|------|
| 参考文献 | 5 篇 | 确定研究方向、方法论框架 |
| 实验室图片 | 20 张 | 充实实验部分、增加真实感 |
| 格式规范 | 无 | 需自行查找或采用通用学术论文格式 |
| 具体要求 | 无（"先写着"） | 需主动建立框架，后续灵活调整 |

### 应对策略
1. **先出框架再填充**：不急于动笔写正文，先搭好完整的论文骨架
2. **以参考文献为锚点**：5 篇文献决定了研究的基调和技术路线
3. **图片反推实验方法**：通过 20 张实验图片推断实验流程和表征手段
4. **补充文献扩大视野**：在 5 篇核心文献基础上再补 15-25 篇，覆盖综述和对比
5. **格式先按国标走**：采用 GB/T 7714 参考文献格式 + 常见本科/硕士论文模板

---

## 二、第一步：解读 5 篇参考文献（最关键）

### 阅读策略
对每篇文献提取以下关键信息，制作一张 **文献信息卡**：

```
文献编号：[1]
标题：
作者/年份：
核心研究内容：（一句话概括）
碳源/原料：
制备方法：（水热法/微波法/溶剂热法/电化学法等）
表征手段：（TEM/XRD/FTIR/UV-Vis/PL 等）
应用方向：（生物成像/离子检测/光催化/LED 等）
关键结论：
可借鉴之处：
```

### 从文献中确定的核心问题
- **制备方法是什么？** → 决定论文实验部分的核心
- **用了什么碳源？** → 绿色碳源（水果、植物）还是化学试剂？
- **表征了哪些性质？** → 决定你需要讨论哪些图片
- **应用在哪个领域？** → 决定论文的应用章节

---

## 三、第二步：分析 20 张实验图片

### 图片分类方法
将 20 张图片按以下类别归类：

| 类别 | 可能包含的图片 | 对应论文章节 |
|------|----------------|--------------|
| **形貌表征** | TEM/SEM/AFM 图像、粒径分布图 | 3.2 形貌与结构表征 |
| **结构表征** | XRD 图谱、FTIR 红外光谱、XPS 图谱 | 3.3 化学结构分析 |
| **光学性质** | UV-Vis 吸收光谱、荧光发射光谱(PL)、荧光照片 | 3.4 光学性能研究 |
| **应用测试** | 检测灵敏度曲线、选择性对比图、细胞成像 | 第四章 应用研究 |
| **实验过程** | 反应装置照片、样品照片（不同颜色溶液） | 2.2 实验方法 |

### 图片信息提取
对每张图片记录：
- 图片编号 + 简要描述
- 推测的表征仪器/实验方法
- 可得出的结论
- 在论文中的位置（章节编号）

---

## 四、第三步：补充参考文献

### 必须补充的文献类型
1. **综述论文 2-3 篇**：碳量子点研究进展综述（用于撰写绑论 1.1-1.3）
2. **同类制备方法 3-5 篇**：与客户实验方法相同的其他研究
3. **同类应用方向 3-5 篇**：与客户应用方向相同的其他研究
4. **表征方法参考 2-3 篇**：用于解释各表征结果的原理

### 推荐检索数据库
- **中文**：知网(CNKI)、万方、维普
- **英文**：Web of Science、Scopus、Google Scholar
- **关键词**：碳量子点 / carbon quantum dots / carbon dots / CQDs / CDs + 制备方法 + 应用方向

---

## 五、第四步：确定论文格式

### 无格式要求时的通用方案

#### 页面设置
- A4 纸，上下左右页边距 2.5cm
- 正文：宋体小四号，1.5 倍行距
- 标题：黑体（一级三号，二级四号，三级小四号加粗）
- 英文和数字：Times New Roman

#### 字数要求（根据学位级别估算）
| 学位 | 建议字数 | 参考文献数量 |
|------|----------|--------------|
| 本科 | 8,000-15,000 字 | 20-30 篇 |
| 硕士 | 30,000-50,000 字 | 50-80 篇 |

#### 参考文献格式
采用 **GB/T 7714-2015** 顺序编码制：
```
[1] 作者. 题名[J]. 刊名, 年, 卷(期): 起止页码.
[2] 作者. 书名[M]. 出版地: 出版社, 年.
```

## 六、写作顺序建议（不要从头写到尾！）

### 推荐写作顺序
```
第一批（有图有数据，最容易写）：
  → 第二章 实验部分（材料、仪器、方法）
  → 第三章 结果与讨论（看图说话）

第二批（需要综合分析）：
  → 第四章 应用研究

第三批（需要文献支撑）：
  → 第一章 绑论（研究背景、文献综述、研究目的）

第四批（最后写）：
  → 摘要（中英文）
  → 结论
  → 致谢
```

### 为什么这个顺序？
- 实验部分最客观，有图有数据，不需要太多主观分析
- 先写完实验才知道该在绪论里铺垫什么
- 摘要是全文精华，必须最后写

## 七、与客户沟通清单

虽然客户说"先写着"，但以下信息最好尽早确认：

### 必须确认
- [ ] 学位级别（本科/硕士/博士）→ 决定字数和深度
- [ ] 学校名称 → 查找该校论文模板
- [ ] 专业方向 → 调整论文侧重点（化学/材料/环境/生物）

### 尽量确认
- [ ] 20 张图片的具体说明（哪些仪器拍的、实验条件）
- [ ] 5 篇文献的全文 PDF（不是只有标题）
- [ ] 是否有原始实验数据（Excel 表格等）
- [ ] 论文查重率要求
- [ ] 交稿时间节点

### 可以后续确认
- [ ] 是否需要开题报告
- [ ] 答辩 PPT 是否需要
- [ ] 修改轮次约定
