# 碳量子点毕业论文 — 快速上手清单

## 拿到手后立即做的 5 件事

### Step 1: 精读 5 篇参考文献（第 1 天）

- [ ] 下载/获取 5 篇文献的全文 PDF
- [ ] 为每篇文献制作信息卡（见写作策略文档）
- [ ] 标记制备方法、碳源、表征手段、应用方向
- [ ] 找出 5 篇文献的共同点 → 这就是论文的核心方向
- [ ] 列出所有引用的参考文献 → 从中挑选补充文献

### Step 2: 分类 20 张实验图片（第 1 天）

- [ ] 将图片按类型归类：形貌/结构/光学/应用/实验过程
- [ ] 给每张图编号并写简要描述
- [ ] 确认使用了哪些表征仪器
- [ ] 推断实验流程和条件
- [ ] 确定每张图在论文中的位置

### Step 3: 确定论文框架（第 2 天）

- [ ] 根据文献和图片确定：碳源 → 制备方法 → 表征 → 应用
- [ ] 调整大纲模板，删去不涉及的部分
- [ ] 确定每章的预估字数
- [ ] 发给客户确认框架

### Step 4: 补充参考文献（第 2-3 天）

- [ ] 在知网/Web of Science 检索相关综述
- [ ] 补充同类制备方法的文献 3-5 篇
- [ ] 补充同类应用方向的文献 3-5 篇
- [ ] 总文献数达到 20-30 篇（本科）或 50+ 篇（硕士）

### Step 5: 开始动笔（从实验部分开始）（第 3 天起）

- [ ] 先写第二章实验部分（最客观，有图有数据）
- [ ] 再写第三章结果与讨论（看图说话）
- [ ] 然后写第四章应用研究
- [ ] 接着写第一章绪论（文献综述）
- [ ] 最后写摘要、结论、致谢

---

## 关键提醒

1. **不要从绪论开始写** — 绪论需要大量文献积累，放到后面写
2. **图片是最大的资产** — 20 张图片足够支撑实验部分
3. **先交框架给客户审核** — 避免方向偏差返工
4. **查重要提前考虑** — 写作时就注意用自己的话表述
5. **格式可以最后统一调** — 内容为王，格式其次
